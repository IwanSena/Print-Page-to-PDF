‚¨áÔ∏è Download Ekstensi (.ZIP)
  1. Klik tombol Download di atas.
  2. Buka File Manager, cari file zip tersebut, lalu Ekstrak (Unzip) menjadi folder.
  3. Buka browser (misal: Kiwi Browser).
  4. Ketik di kolom alamat: chrome://extensions
  5. Aktifkan Mode Pengembang (Developer Mode).
  6. Klik tombol + (Load Unpacked / Muat).
  7. Pilih folder hasil ekstrak tadi. Selesai!

‚öôÔ∏è Penjelasan Fitur

1. Mode Bersih (Hapus Iklan)

Fitur ini otomatis mendeteksi dan menghapus iklan, tombol-tombol mengganggu, dan menu navigasi sebelum dicetak. Hasil PDF jadi lebih rapi dan enak dibaca.

2. Satu Halaman Panjang (Tanpa Potongan)

Membuat PDF menjadi satu lembar panjang ke bawah (seperti screenshot panjang).

Penting: Saat mode ini aktif, pilihan "Ukuran Kertas" (A4/A5) tidak berlaku karena ukuran kertas akan menyesuaikan panjang website secara otomatis.

‚ùì Tanya Jawab (Q&A)

Q: Mengapa download gagal atau tidak berjalan? A: Cobalah untuk memuat ulang (Refresh) halaman website. Terkadang memori browser penuh dan butuh penyegaran agar skrip pencetak berjalan lancar.
Q: Masih gagal download setelah refresh? A: Coba ubah tampilan browser Anda. Jika sedang di Tampilan Ponsel, ubah ke Situs Desktop, atau sebaliknya. Beberapa website memiliki struktur yang sulit dibaca di mode tertentu.
Q: Apa bedanya dengan fitur bawaan browser? A: Fitur bawaan sering error saat proses spooling atau hasilnya terpotong. Ekstensi ini memproses langsung di memori dan menyimpannya ke folder Download Internal. Tingkat keberhasilan hampir 100%.
Q: Apakah data saya aman? A: 100% Aman. Ekstensi ini bekerja lokal (offline-capable) di HP Anda. Kami tidak punya server, jadi tidak ada data browsing yang dikirim keluar.
