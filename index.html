  1. Buka halaman web yang ingin Anda simpan (misal: Chord gitar, Artikel berita).
  2. Tunggu sampai halaman selesai dimuat (loading selesai).
  3. Buka menu browser, cari dan klik Cetak Halaman ke PDF.
  4. Pilih opsi yang diinginkan, lalu tekan tombol Unduh.
  5. Tunggu proses berjalan (lihat status di bawah tombol), PDF akan terunduh otomatis.

1. Mode Bersih (Hapus Iklan)

Fitur ini otomatis mendeteksi dan menghapus iklan, tombol-tombol mengganggu, dan menu navigasi sebelum dicetak. Hasil PDF jadi lebih rapi dan enak dibaca.

2. Satu Halaman Panjang (Tanpa Potongan)

Membuat PDF menjadi satu lembar panjang ke bawah (seperti screenshot panjang).

Penting: Saat mode ini aktif, pilihan "Ukuran Kertas" (A4/A5) tidak berlaku karena ukuran kertas akan menyesuaikan panjang website secara otomatis.

üîí Kebijakan Privasi (Privacy Policy)

Keamanan data Anda adalah prioritas kami.

  • 100% Lokal: Proses pembuatan PDF terjadi sepenuhnya di dalam memori HP/Browser Anda.
  • Tanpa Server: Kami tidak mengirimkan URL, isi halaman, atau data apapun ke server luar.
  • Tanpa Pelacak: Ekstensi ini tidak memata-matai aktivitas browsing Anda.
